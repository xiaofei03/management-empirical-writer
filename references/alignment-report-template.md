# Alignment Report Template

Use this template after the intake package is complete and before any drafting begins.

The user must explicitly confirm this report before Stage 2 starts.

## Output Rules

- Be specific and evidence-based.
- Distinguish confirmed facts from tentative inference.
- List unresolved questions clearly.
- Do not hide uncertainty.

## Reporting Guidance

### Mark evidence status explicitly

Use labels such as:

- `Confirmed from user materials`
- `High-confidence inference from results and notes`
- `Needs user confirmation`

### Typical confirmation questions

- Is the dependent variable definition stated exactly right?
- Is the core mechanism framed as mediation or as theoretical explanation only?
- Is the fixed-effects structure complete?
- Is the clustering level final?
- Should the English version target a general management journal or a narrower strategy or governance audience?

## Failure Rule

If the available evidence is too thin to fill this report with confidence:

- Stop
- Return a shorter diagnostic alignment memo
- Ask for the missing materials

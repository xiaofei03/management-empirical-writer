# management-empirical-writer

Reusable workflow for evidence-grounded bilingual empirical management manuscripts.

It owns alignment, drafting, internal review, Chinese-English equivalence, the transition from Markdown-first drafting to Word-only refinement, and freezing a generic submission mother. It delegates target selection, journal adaptation, empirical computation, and final Word geometry to their specialist skills.

The project uses canonical drafts, one active `_work/<run_id>/` workspace, one cumulative revision ledger, named frozen milestones, and Git history. Numbered Word rounds and root-level process artifacts are prohibited.
